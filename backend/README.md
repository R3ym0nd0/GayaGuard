# Backend Structure

This folder is prepared for the backend of the Barangay Gaya Gaya Request Screening and Monitoring System.

Suggested usage:

- `src/config` for environment and database setup
- `src/controllers` for request handlers
- `src/middleware` for auth and validation middleware
- `src/models` for database models
- `src/routes` for API routes
- `src/services` for business logic
- `src/utils` for shared helpers
- `uploads` for user-submitted files
